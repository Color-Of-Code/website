# Set Top Box GBUE 4K

## Hardware

Important Features:

* GigaBlue UHD UE 4K
* 1x [Full Band Capture](https://wiki.openpli.org/Full_Band_Capture) DVB-C NIM 3128 FBC (8 tuner)
* 1x OUT Loop through
* Broadcom 7252s
* RAM 2GB DDR4
* FLASH 4GB MMC
* 1x HDMI out
* 1 GB LAN
* Front: 1x USB 2.0
* Rear: 1x USB 3.0 2x USB 2.0
* 1x CI Slot
* 2x SmartCard Slot

## Software

* Open ATV 6.3 ([Images](http://images.mynonpublic.com/openatv/current/index.php?open=gbue4k))
* OE-Alliance 4.3
* Linux 4.1.20 / 20200312
* MultiBoot: 3 Slots
